# express-post-json-quiz-notes

## Quiz Questions

After completing this exercise, you should be able to discuss or answer the following questions:

- What does the `express.json()` middleware do and when would you need it?
  the express.json() middleware is used to parse incoming JSON data in the req.body object, which can then be used in the /api/data route handler to process and respond to the JSON data.
